Get Started

    
Install via npm or use directly in the browser with unpkg.

    

      $ npm install @decimalturn/toml-patch
    

    
Or use in the browser with a module script:

    

      <script type="module">

        import * as TOML from

          'https://unpkg.com/@decimalturn/toml-patch';

      </script>
    

    

      Key features: comment & formatting preservation during patches,
      full TOML v1.1 support, zero dependencies, works in Node.js and browsers.
